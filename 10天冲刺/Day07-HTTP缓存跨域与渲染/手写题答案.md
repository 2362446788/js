# Day 7 手写题答案

## 题 1：实现 `ajax(options)`

来源：

- 2026 Front End Interview Handbook Ajax
- 飞书《大前端面试宝典》Q69

```js
function ajax(options) {
  const { method = "GET", url, data = null, timeout = 0 } = options;

  return new Promise((resolve, reject) => {
    const xhr = new XMLHttpRequest();
    xhr.open(method, url, true);
    xhr.timeout = timeout;

    xhr.onreadystatechange = function () {
      if (xhr.readyState !== 4) return;
      if (xhr.status >= 200 && xhr.status < 300) {
        resolve(xhr.responseText);
      } else {
        reject(new Error(`Request failed: ${xhr.status}`));
      }
    };

    xhr.ontimeout = function () {
      reject(new Error("Request timeout"));
    };

    xhr.onerror = function () {
      reject(new Error("Network error"));
    };

    if (data && method.toUpperCase() !== "GET") {
      xhr.setRequestHeader("Content-Type", "application/json");
      xhr.send(JSON.stringify(data));
    } else {
      xhr.send();
    }
  });
}
```

详细思路：

1. 创建 `XMLHttpRequest`
2. 配置请求方法、地址、超时
3. 监听 `readystatechange`
4. 状态完成后根据状态码判断成功或失败
5. 额外监听超时和网络错误

复杂度：

- 逻辑复杂度不高，主要考浏览器 API 熟悉度

## 题 2：实现 `jsonp`

来源：

- 飞书《大前端面试宝典》Q71
- Day07 跨域专题

```js
function jsonp(url, params = {}, callbackKey = "callback") {
  return new Promise((resolve, reject) => {
    const callbackName = `jsonp_${Date.now()}_${Math.random().toString(16).slice(2)}`;
    const script = document.createElement("script");
    const query = new URLSearchParams({
      ...params,
      [callbackKey]: callbackName,
    });

    window[callbackName] = function (data) {
      resolve(data);
      cleanup();
    };

    script.onerror = function () {
      reject(new Error("JSONP request failed"));
      cleanup();
    };

    function cleanup() {
      delete window[callbackName];
      script.remove();
    }

    script.src = `${url}${url.includes("?") ? "&" : "?"}${query.toString()}`;
    document.body.appendChild(script);
  });
}
```

详细思路：

1. 生成一个全局唯一回调名
2. 把回调名拼到 URL 参数里
3. 动态创建 `script`
4. 服务端响应后执行全局函数，把数据传回来
5. 执行结束后清理脚本和全局函数

易错点：

- 只适用于 GET
- 必须做清理，否则容易污染全局

## 题 3：实现 `delegate`

来源：

- 2026 Front End Interview Handbook Event delegation
- 飞书《大前端面试宝典》Q114

```js
function delegate(parent, selector, eventName, handler) {
  function listener(event) {
    let target = event.target;
    while (target && target !== parent) {
      if (target.matches(selector)) {
        handler.call(target, event);
        break;
      }
      target = target.parentElement;
    }
  }

  parent.addEventListener(eventName, listener);
  return () => parent.removeEventListener(eventName, listener);
}
```

详细思路：

1. 事件绑定在父节点上
2. 触发时，从 `event.target` 开始向上找
3. 找到匹配 `selector` 的元素就执行回调
4. 没找到就继续向父层走，直到 `parent`

为什么返回解绑函数：

- 方便业务层在组件卸载时清理事件

## 题 4：实现 `getElementsByClassNameLite`

来源：

- 2026 Front End Interview Handbook getElementsByClassName

```js
function getElementsByClassNameLite(root, className) {
  const result = [];

  function dfs(node) {
    if (node.nodeType !== 1) return;
    if (node.classList && node.classList.contains(className)) {
      result.push(node);
    }
    for (const child of node.children) {
      dfs(child);
    }
  }

  dfs(root);
  return result;
}
```

详细思路：

1. 深度优先遍历整棵 DOM 树
2. 只处理元素节点
3. 用 `classList.contains` 判断是否命中目标类名
4. 命中就收集到结果数组

复杂度：

- 时间复杂度：`O(n)`，`n` 是节点数量
- 空间复杂度：`O(n)`，极端情况下结果很多
