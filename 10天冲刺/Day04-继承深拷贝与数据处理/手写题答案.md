# Day 4 手写题答案

## 题 1：实现 `deepClone`

来源：

- 2026 Front End Interview Handbook Deep Clone
- 高频前端手写题

```js
function deepClone(value, cache = new WeakMap()) {
  if (value === null || typeof value !== "object") return value;
  if (cache.has(value)) return cache.get(value);

  if (value instanceof Date) return new Date(value);
  if (value instanceof RegExp) return new RegExp(value.source, value.flags);

  if (value instanceof Map) {
    const result = new Map();
    cache.set(value, result);
    value.forEach((v, k) => {
      result.set(deepClone(k, cache), deepClone(v, cache));
    });
    return result;
  }

  if (value instanceof Set) {
    const result = new Set();
    cache.set(value, result);
    value.forEach((item) => {
      result.add(deepClone(item, cache));
    });
    return result;
  }

  const result = Array.isArray(value) ? [] : {};
  cache.set(value, result);

  Reflect.ownKeys(value).forEach((key) => {
    result[key] = deepClone(value[key], cache);
  });

  return result;
}
```

详细思路：

1. 原始值直接返回，因为它们没有“引用层”
2. 循环引用先查 `WeakMap` 缓存，避免递归死循环
3. 特殊对象单独处理：
  - `Date`
  - `RegExp`
  - `Map`
  - `Set`
4. 普通对象和数组递归拷贝所有自有 key

为什么要用 `WeakMap`：

- key 必须是对象
- 不会阻止垃圾回收

复杂度：

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 题 2：实现 `flatten`

来源：

- 2026 Front End Interview Handbook Flatten
- 高频前端手写题

```js
function flatten(arr) {
  return arr.reduce((acc, item) => {
    if (Array.isArray(item)) {
      return acc.concat(flatten(item));
    }
    acc.push(item);
    return acc;
  }, []);
}
```

详细思路：

1. 遍历数组每一项
2. 如果当前项还是数组，就递归拍平
3. 如果不是数组，就直接放进结果

复杂度：

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 题 3：实现 `uniqueBy(arr, key)`

来源：

- 飞书《大前端面试宝典》Q96
- 高频数据处理题

```js
function uniqueBy(arr, key) {
  const seen = new Map();
  return arr.filter((item) => {
    const value = typeof key === "function" ? key(item) : item[key];
    if (seen.has(value)) return false;
    seen.set(value, true);
    return true;
  });
}
```

详细思路：

1. 用 `Map` 记录已经见过的分组值
2. 每次取出当前项的判重值
3. 如果已经存在，就过滤掉
4. 如果没存在，就保留并记录

复杂度：

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`

## 题 4：实现 `listToTree(list)`

来源：

- 高频前端数据结构手写题

```js
function listToTree(list) {
  const map = new Map();
  const result = [];

  list.forEach((item) => {
    map.set(item.id, { ...item, children: [] });
  });

  list.forEach((item) => {
    const node = map.get(item.id);
    if (item.parentId == null) {
      result.push(node);
    } else {
      const parent = map.get(item.parentId);
      if (parent) parent.children.push(node);
    }
  });

  return result;
}
```

详细思路：

1. 第一次遍历先建 `id -> node` 的映射表
2. 第二次遍历根据 `parentId` 把节点挂到父节点的 `children`
3. 没有 `parentId` 的节点就是根节点

为什么不用每次都去全表找父节点：

- 那样会退化成 `O(n^2)`
- 建 map 后可以优化到 `O(n)`

## 题 5：实现 `groupBy(arr, key)`

来源：

- 2026 Front End Interview Handbook Lodash groupBy examples

```js
function groupBy(arr, key) {
  return arr.reduce((acc, item) => {
    const groupKey = typeof key === "function" ? key(item) : item[key];
    if (!acc[groupKey]) acc[groupKey] = [];
    acc[groupKey].push(item);
    return acc;
  }, {});
}
```

详细思路：

1. 用 `reduce` 把数组累积成对象
2. 每次先算当前元素分到哪一组
3. 如果这一组还不存在，就先初始化为空数组
4. 再把当前元素 push 进去

复杂度：

- 时间复杂度：`O(n)`
- 空间复杂度：`O(n)`
